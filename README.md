# OpenGL Valgut Engine
Early development of a basic 3D engine built from scratch.

## Current State
Getting the foundations in place — rendering pipeline, camera controls, and a debug UI using ImGui.

## Stack
- OpenGL 3.3
- GLFW
- GLM
- ImGui

## What's in so far
- Phong lighting (ambient, diffuse, specular)
- Stencil outline on objects
- WASD + mouse to move around
- TAB to switch between camera and UI
- Debug grid
- ImGui side panels with controls for size, position, color, outline and lighting

## Status
Work in progress. Nothing stable yet.
